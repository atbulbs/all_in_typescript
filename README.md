A webpack config in TypeScript. Async import, code split and tree shaking are supported. Welcome star:star:  . Happy Hacking :smile:

# all_in_typescript
```bash
$ git clone https://github.com/atbulbs/all_in_typescript.git
$ cd all_in_typescript
$ npm i
$ npm start
```
